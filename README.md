# MovieVibe 🎬
**AI-Powered Movie Recommendations Based on Your Mood**

MovieVibe is a modern web application that helps you find the perfect movie to watch by simply describing your current "vibe." Whether you had a long day at work, you're feeling nostalgic for the 90s, or you want something that will make you think, our AI understands your feelings and provides tailored suggestions.

---

## ✨ Features

- **Vibe-Based Discovery**: No more scrolling through endless lists. Just type how you feel.
- **AI Reasoning**: Get personalized explanations on *why* a movie fits your specific mood.
- **Dynamic Movie Posters**: High-quality visual experience with smart fallback systems (Metahub & iTunes).
- **Modern Interface**: A sleek, dark-themed UI built with Tailwind CSS and smooth animations.
- **Full-Stack Architecture**: Powered by a robust Node.js/Express backend and a React frontend.
- **OpenAI Integration**: Uses OpenAI's advanced language models for intelligent recommendations.

---

## 🚀 Tech Stack

- **Frontend**: React 19, Tailwind CSS 4, Motion (Framer Motion), Lucide Icons
- **Backend**: Node.js, Express 5
- **AI Engine**: OpenAI API (GPT-4o-mini)
- **Data Sources**: iTunes API & Metahub for high-res posters

---

## 📝 How It Works

1. **Input**: You enter a description like *"I want a mind-bending sci-fi movie that feels like a dream."*
2. **AI Processing**: The app sends your prompt to the backend, which communicates with the OpenAI API.
3. **Analysis**: The AI selects the best matches and writes a custom recommendation for the top pick.
4. **Result**: You get a beautiful display of movies with posters, ratings, and summaries.
